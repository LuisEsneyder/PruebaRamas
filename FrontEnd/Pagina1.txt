Pasos:
1) Crear carpeta vacia
2Creo repositorio local: git init
3)conecto al repositorio github: git remote add nombreRepo urlRepo
4)creo las ramas en el repositorio local que esten en el repositorio de github: git branch nombre_Rama (Se hace el número de veces que sea necesario para satifacer el numero de ramas) 
5)Selecciono la rama del repositorio local que quiero trar del repositorio remoto: git checkout nombre_Rama
5)traigo con pull lo que este en la rama que quiero traer: git pull nombreRepo nombre_Rama
6)creo un commit antes agrego al area de preparación todo: (git add .), (git commit .m "lo que se quiera escribir")
10)selecciono la rama que quiero mandar al repositorio remoto: git checkout nombre_Rama
11)mando el commit creado en el paso 6 a la rama:git push nombreRepo nombre_Rama
